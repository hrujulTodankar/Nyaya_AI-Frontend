## üîß Usage Guidelines

### Minimum Sizes
- logo.svg: 48px minimum

- logo-mark.svg: 32px minimum
- app-icon.svg: 128px minimum

### Background Recommendations
- Dark backgrounds: Use as-is
- Light backgrounds: Add dark overlay or adjust colors
- Transparent: Works best on dark surfaces

### Don'ts
- Don't change stroke widths
- Don't alter gradient colors
- Don't add text inside logo
- Don't distort proportions
- Don't use on busy backgrounds
